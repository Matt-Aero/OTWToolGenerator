About

OTW Tool Generator (OTG) streamlines the tooling design process for for welded tube assemblies.

Rapidly produce production-ready STEP files with tab+slot features, adjustable tolerancing, and material thickness. Simply input the location and vector of each end point and press go.

Thank you for using OTW Tool Generator during its development. If you have feedback, feature requests, or need help please reach out to otwtoolgenerator@gmail.com.

Example

This tutorial walks through how to use OTG with a simple ¼” flared tubing assembly, containing a welded tee.

1. Position Assembly

From your CAD software, position the OTW assembly above the XY plane. The XY plane represents the workbench or tabletop for the tooling. The Z axis should point upwards, opposite the gravity vector.

2. Selecting Joints

Joints are end points of the tubing assembly that need to be constrained. Each joint consists of a location, pointing vector, and joint type.

The joint location and vector can be easily selected by pressing "Select Location" and then selecting the edge on the assembly that you wish to place a joint at. Alternatively, joint data can be entered in millimeters into the input boxes.

After selecting a joint location, a green vector arrow will appear. The vector arrow points in the direction that the tooling will be extruded. The arrow should face away from the face you have selected for your joint to prevent interferences between the tool and your tubing assembly. The joint vector can be easily rotated 180 degrees by pressing the double arrow button next to the vector input row.

The joint data for this sample assembly is shown in the table below:

Joint # Location (X,Y,Z) [mm] Vector (X,Y,Z)
1 97.625, -72.098, 160.000 0, -1, 0
2 -97.625, -72.098, 160.000 0, -1, 0
3 -35.569, 150.354, 126.050 -7.519, 11.146, -7.176

3. Input Data

Fill out the form data for each joint. Flared joints can be selected of flanged connection with adjustable bolt patterns. Note that each joint on the left panel cooresponds to a joint on the tubing assembly. Loading of the tubing assembly is not required to use the tool but is highly recommended. Loading the tubing assembly allows for inputting joint location and vector data simply by selecting edge on the assembly., input the joint location and vector data from above. Additionally, the exported .stl file of the assembly can be loaded for visualization; however, this is optional.

4. Configure Parameters

Configure the “tab+slot clearance” and “material thickness” parameters. Tab+slot clearance defines the gap in millimeters all around the material tabs. This can help to ensure waterjet and/or 3D printed assemblies fit together. The default value is 0.25mm.

5. Generate Tooling and Export

Press “Generate Tooling” to produce the production-ready tooling part files. If you are happy with the result, press "Export" to download the tooling part files to your local computer.

What’s Next?

After generating the part files, you can send them directly to a manufacturing house to be waterjet or laser-cut. Additionally, you can import the parts into your CAD software and assign them part numbers for tracking in your own part modeling environment. Due to the tab+slot geometry, welding out the final assembly is easy.
