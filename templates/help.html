OTW Tool Generator (OTG) aims to streamline tooling creation for orbital tube welded assemblies and reduce modeling errors leading to costly go-backs during manufacturing.

When tubing is welded to tees, spools, or other tubes the clocking of the weld joint oftentimes needs to be controlled. This is where tooling is used to constrain the end points of the tubing assembly during the welding process.

OTW Tool Generator leaves the high-level design of the tool in the designer’s hands, while rapidly producing production-ready .step files with tab+slot features, adjustable tolerancing, and material thickness. Simply input the location and vector of each end point and press go. OTG generates flat .step part files that can be sent to a waterjet or laser-cutting house and then assembled.

Thank you for using OTW Tool Generator during its development. If you have any feedback or need help, please reach out to otwtoolgenerator@gmail.com.

Workflow Example

This tutorial walks through how to use OTG with a simple ¼” flared tubing assembly, containing a welded tee. Note: OTG only supports ¼, ½, and 1 flared joint assemblies at this time.

1. From your favorite CAD software, position the OTW assembly above the XY plane. The XY plane representing the workbench or tabletop for the tooling. The Z axis should point upwards, opposite the gravity vector. Note that you may need to add suitable end fittings or constraints to interface with the tubing. Shown below are through-wall fittings (PN: 50715K581 from McMaster-Carr) which have extended threaded shafts for fitting through panels.

2. Record the location (X,Y,Z) and pointing vector (X,Y,Z) of each of the desired joint locations. The location should be measured in millimeters at the center of the face where you want the tool placed. In this example, the location is measured on the end fittings, near the hex face.

The joint data for this sample assembly is shown in the table below:

Joint # Location (X,Y,Z) [mm] Vector (X,Y,Z)
1 97.625, -72.098, 160.000 0, -1, 0
2 -97.625, -72.098, 160.000 0, -1, 0
3 -35.569, 150.354, 126.050 -7.519, 11.146, -7.176

3. (Optional) Export the tubing assembly as an .stl file for visualization in the OTG web-application.

4. In the OTW Tool Generator web application, input the joint location and vector data from above. Additionally, the exported .stl file of the assembly can be loaded for visualization, however, this is optional.

5. Configure the “tab+slot clearance” and “material thickness” parameters. Tab+slot clearance defines the gap in millimeters all around the material tabs. This can help to ensure waterjet and/or 3D printed assemblies fit together. The default value is 0.25mm.

6. Press “Generate Tooling” to see generated files displayed on screen. The export button can be used to export the files.

What’s next?

After generating the part files, you can send them directly to a manufacturing house to be waterjet or laser-cut. Additionally, you can import the parts into your CAD software and assign them part numbers for tracking in your own part modeling environment. Due to the tab+slot geometry, welding out the final assembly is easy.
